# 📄 Pagination App

This is an application designed to paginate any PDF document, supporting both numbers and letters for pagination.

## 🛠️ Built With

- [Poetry](https://python-poetry.org/) – Python dependency management and packaging tool.

## 🚀 Getting Started

To run this project, follow the steps below:

### 1. Navigate to the Project Directory

Make sure you're inside the folder where the `pyproject.toml` file is located.

### 2. Remove the Lock File (if exists)

```bash
rm poetry.lock
```

### 3. Install Dependencies

```bash
poetry install
```

### 4. Activate the Virtual Environment

```bash
poetry shell
```

### 5. Run the Application

```bash
python your_python_file.py
```

## 🧾 Features

- Paginate PDF documents using custom sequences.
- Supports numeric and alphabetic pagination styles.

## 📌 Note

Make sure you have Poetry installed. If not, you can install it by following the instructions at [python-poetry.org/docs](https://python-poetry.org/docs/).
